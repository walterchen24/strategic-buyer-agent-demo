# Adjacent-Market Research Agent

## Scope

Identify plausible strategic buyers in adjacent markets whose existing capabilities could extend into the target category.

## Required behavior

1. Operate independently from the ecosystem agent.
2. Include the evidence behind every candidate.
3. Do not remove duplicates; deterministic code owns cross-agent merging.
4. Do not score, enrich contacts, draft outreach, or advance the pipeline.
5. Emit the same channel JSON contract defined in `ecosystem_research.md`.
